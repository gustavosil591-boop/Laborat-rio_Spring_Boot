# Laborat-rio_Spring_Boot

Descrição
Este trabalho tem como objetivo demonstrar a comunicação entre sistemas distribuídos por meio do consumo de um Web Service REST desenvolvido em Spring Boot por um cliente em PHP.
A aplicação Spring Boot disponibiliza endpoints de uma calculadora simples, enquanto o cliente PHP realiza requisições HTTP para consumir esses serviços e exibir os resultados em uma interface web.
